=== "Grammar"

    ## Numbers
    
    Today we will learn about numbers. Here's how to count from 1 to 3:
    
    ```
    tam = 1
    tor = 2
    par = 3
    ```
    
    So if you want to say 1, 2, 3, you'd say tam, tor, par.
    
    Now count backwards from 3 to 1. 
    
    Great! The numbers will help us talk about family members. That's because family in Oravia is like a map of the family tree. Last lessons, we learned that:
    
    | Oravia | English |
    |--------|---------|
    | fare | parent |
    | fasu | sibling|
    | fano | son, daughter |
    
    These are actually indicating what is the relation to the point of reference.
    
    ```
    re -> up
    su -> side
    no -> down
    ```
    
    So if you think of a family tree and the generations, we would have:
    ```
    parent = up (re)
    sibling = side (su)
    child = down (no)
    ```
    
    Using the numbers, here is how we say grandparent:
    
    ```
    fa    +   tor  +  re 
    ↓          ↓       ↓ 
    family    two     up
    member
    ```
    
    *Fatore* is grandparent! It means twice up.
    
    This is how we say cousin:
    
    ```
    fa    +   tor  +  su 
    ↓          ↓       ↓ 
    family    two     side
    member
    ```
    
    *Fatorsu* is cousin! It means twice to the side.
    
    This is how we say aunt or uncle:
    
    ```
    fa    +    re  +   su
    ↓          ↓       ↓ 
    family     up    side
    member
    ```
    
    *Faresu* is aunt/uncle! It means going up, and then sideways. If you prefer to think about it another way, it's parent sibling. 
    
    Can you guess how we would say *granddaughter*? How about *nephew*? And greatgrandfather?

    To sum it up:
    ```
    tam = one
    tor = two
    par = three
    parent = up (re)
    subling = side (su)
    child = down (no)
    ```
    
    Now you can talk about pretty much any family member! If you want to talk about in-laws, you could, for example, use *faigel* (marriage) in the front as a modifierr.

    In sum, in the BO cluster (house), we have:
    
    ```
    BOR - structural  
    BOS - bathroom   
    BON - furniture
    BOE - rooms / locations
    ```
    
    We learned an interesting word here, *boemo*, which means kitchen. Do you notice anything in this word? You know what MO means, and we just learned BOE. Wait a minute...
    
    ```
    boe +  mo  = kitchen
     ↓     ↓                
    room  food  
    
    ```
    
    !!! tip "Sound Connections"
    
    Yes, that's right! Kitchen means *food room*. And *boemo* is actually not a special word: many are like this! That's because not only the first syllables have meaning: others can, too. The more words you learn, the more you discover the syllable-meaning associations, and the easier it gets! At some point not too long from now, you may be guessing the meaning of words you haven't seen before and creating your own words. :)
    
    Ps. Other words you've already learned are also composed of sounds that have meaning in Oravia, just like *boemo*. You knew what MO means, but you may not know what the meanings of these other sounds are yet. As you learn more words, these syllable-meaning associations will become intuitive. I won't be telling you what these associations are: this is for you to discover as you learn more vocabulary. Happy exploration!

    // Load both exercise 1 and exercise 2 words
    Promise.all([
        fetch(baseUrl + '/data/lesson06_words.json').then(r => r.json()),
        fetch(baseUrl + '/data/lesson06_exercise2_words.json').then(r => r.json())
    ])
    .then(results => {
        // Combine words from both exercises
        const allWords = [...results[0].words, ...results[1].words];
        const wrongWords = allWords.filter(word => wrongIds.includes(word.id));
        
        if (wrongWords.length === 0) {
            container.innerHTML = '<div style="text-align: center; padding: 3rem; background: #e0f2f1; border-radius: 8px;"><p style="font-size: 1.2rem; color: #4a9cd6; margin: 0;">🎉 No words to review!</p></div>';
            return;
        }

        container.innerHTML = '<p style="text-align: center; margin-bottom: 2rem; color: #5a8bb8;">Practice these ' + wrongWords.length + ' word(s) you found challenging:</p><div id="review-game-wrapper"></div><div style="text-align: center; margin-top: 2rem;"><button id="clear-review" style="padding: 0.5rem 1.5rem; background: #f57c00; color: white; border: none; border-radius: 4px; cursor: pointer; font-size: 0.95rem;">Clear Review List</button></div>';

        new MatchingGame('review-game-wrapper', wrongWords, 'review', null, []);

        document.getElementById('clear-review').addEventListener('click', function() {
            if (confirm('Clear all review words? This will reset your wrong words list for this lesson.')) {
                const allWrongIds = JSON.parse(localStorage.getItem('wrong_ids') || '[]');
                const lessonWordIds = allWords.map(w => w.id);
                const remainingWrongIds = allWrongIds.filter(id => !lessonWordIds.includes(id));
                localStorage.setItem('wrong_ids', JSON.stringify(remainingWrongIds));
                location.reload();
            }
        });
    })
    .catch(error => {
        console.error('Error loading words:', error);
        container.innerHTML = '<p style="color: #f44336;">Error loading review words. Please refresh the page.</p>';
    });
});
</script>
